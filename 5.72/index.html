Celery的Web监控管理--Flower

Mon, Feb 5, 2018

Flower是Celery的一个实时监控和管理Web界面工具，目前仍在活跃的开发之中，但已经是一个很重要的可用工具了。这是推荐使用的Celery监控工具。

1，安装依赖

pip install flower

2，在项目目录下运行 flower命令启动web-server

python manage.py celery flower

3，通过浏览器访问flower服务器

http://localhost:5555
