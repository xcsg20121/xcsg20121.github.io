两种常见判断变量的方法

Sun, Apr 15, 2018

两种常见判断变量的方法（同样适用于其他的数据类型）

demo: 　# 判断变量是否为整形

a = 5
if isinstance(a,int):
    print("%d是整形"%a)
else:
    print("%s不是整形"%a)

a = "5"
if type(a) == int:
    print("%d是整形"%a)
else:
    print("%s不是整形"%a)


#运行结果
5是整形
5不是整形
