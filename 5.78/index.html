Http-server 的使用

Fri, Jun 1, 2018

Http-server 是一款基于node.js的web前端开发服务，可以很好的承担前后端解耦后，前端服务的搭建。

1，首先安装node

node下载地址：https://nodejs.org/zh-cn/

2，在命名行执行

npm install http-server –g

3在前端项目目录启动服务（hs）
