Django 配置Mysql数据库

Tue, Nov 10, 2015

首先在setting文件数据库配置中设置

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': "mydjango",
        'USER':"root",
        "PASSWORD":"mysql",
        # ''空位默认
        'HOST':'',
        'PORT':'',
        'OPTIONS':{'autocommit':True}

    }
}

然后在init.py中初始化mysql()类

import pymysql
pymysql.install_as_MySQLdb()

其次在项目根目录中设置初始化（命令提示窗）

python manage.py makemigrations
python manage.py migrate
